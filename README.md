# 项目简介

Avision AI 是一个帮助用户设计和装修的智能平台。通过简单的问答，系统能够理解用户的喜好并提供个性化的设计建议。

## 安装步骤

1. 克隆项目到本地：
   ```bash
   git clone <repository-url>
   ```
2. 进入项目目录：
   ```bash
   cd weborigin
   ```
3. 安装依赖：
   ```bash
   npm install
   ```
4. 启动开发服务器：
   ```bash
   npm run dev
   ```

## 使用说明

- 访问 `http://localhost:3000` 查看项目。
- 根据需要修改配置文件以适应不同的环境。
